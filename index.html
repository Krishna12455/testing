VISUAL STUDIO CODE

Visual Studio Code, commonly referred to as VS Code is an integrated development environment developed by Microsoft for Windows, Linux, macOS and web browsers. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded version control with Git. Users can change the theme, keyboard shortcuts, preferences, and install extensions that add functionality.

Visual Studio Code is proprietary software released under the "Microsoft Software License", but based on the MIT licensed program named "Visual Studio Code-Open Source" (also known as "Code-OSS"), also created by Microsoft and available through GitHub. Open-source binaries are available from VSCodium, without Microsoft's telemetry.

In the Stack Overflow 2023 Developer Survey, Visual Studio Code was ranked the most popular developer environment tool among 86,544 respondents, with 73.71% reporting that they use of Visual Studio.

KEY FEATURES VISUAL STUDIO CODE

  • Cross-Platform : VS Code runs on Windows, macOS, and Linux, making it accessible to developers across different operating systems

  • Support for Multiple Languages : It supports many programming languages out of the box, like JavaScript, Python, C++, and HTML, and through extensions, you can add support for more languages such as Rust, Go, and even niche languages.

  • Intelligent Code Editing :

  1. Syntax Highlighting : Colors keywords, variables, and syntax to make code easier to read.

  2. IntelliSense : Provides code completions, parameter suggestions, and quick info about functions or methods

  3. Auto-formatting : Automatically formats code according to set guidelines, keeping your code clean.

  • Debugging : VS Code has built-in debugging tools, which allow you to debug code directly within the editor. It supports breakpoints, call stacks, and an interactive debugging console.

  • Extensions and Integrations : The marketplace offers thousands of extensions to add new features and tools to VS Code, such as linters, themes, version control, Docker support, and database tools.

  • Version Control : It has integrated Git support, allowing you to clone repositories, manage branches, and commit code right from the editor.

  • Terminal : VS Code includes an integrated terminal, so you can run shell commands without leaving the editor.

  • Customization : You can customize VS Code’s appearance and behavior with themes, settings, and keybindings. You can also install various extensions to tailor the editor to your workflow.

  • Live Share : With the Live Share extension, you can collaborate in real-time with other developers on the same codebase.

  • Performance : It’s known for being lightweight, which makes it ideal for fast startups and efficient performance, even on lower-end systems

EMMETS

Emmet is a plugin built into modern code editors like VS Code that helps developers write HTML and CSS faster. It's essentially a shorthand system for quickly generating HTML and CSS code using abbreviations. When you type an Emmet abbreviation and expand it (usually by pressing Tab or Enter), it generates a full HTML structure or CSS rule

HTML Emmets Description
> Creates a parent-child relationship between elements
+ Adds a sibling element to the current element.
^ Moves up one level to the parent element.
* Repeats the current element a specified number of times.
{} Adds custom text inside an element.
[] Adds attributes to an element.
() Groups elements together to apply operations like multiplication or nesting.
> and + Create complex structures using a combination of child and sibling selectors.
! Generates a basic HTML5 document structure.
| Used to add incremental numbers in repeated elements.
@ Specifies the starting index for numbered items.
